# Reading is Good
Developed by Hakan Arayici in Intellij IDEA

When you run project Application is served by using 8081 port

## Project setup
```
Build the project using mvn clean install
Run using mvn spring-boot:run
```

## To Create and Run Docker Container
```
docker build -t readingisgoodapp .
docker run -p 8081:8081 -d readingisgoodapp
```

## API Guide
```
You can reach api guide generated by ascii doc at [http://localhost:8081/docs/api-guide.html]
```

## Swagger
```
You can reach api guide generated by swagger open api at [http://localhost:8081/swagger-ui.html]
Be aware of : You can not call api directly by using swagger because of authentication header is required
```

## POSTMAN
```
Postman collection is available under postman directory
You can import and call the api. All API's must include 
Header Authorization bearer token which can be obtained by
calling http://localhost:8081/authenticate api
username : hakanarayici
password : password
```

## Reaching DB Console
```
You can reach H2 database console at [http://localhost:8081/h2/]
With default username sa without using password
```
